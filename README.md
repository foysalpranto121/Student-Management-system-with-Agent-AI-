# 🎓 Student Management System with AI Assistant

A modern web-based academic management system enhanced with an intelligent AI assistant for analytics, predictions, and smart data interaction.

---

## 🚀 Overview

This project combines traditional student management functionalities with advanced AI capabilities to help institutions manage data efficiently and gain meaningful insights from it.

---

## ✨ Features

### 📚 Core Academic Management
- Role-based access control (Admin & Teacher)
- Full student CRUD operations
- Subject-wise grading and GPA calculation
- Attendance tracking and monitoring
- PDF report generation using `reportlab`

### 🤖 AI Assistant (Advanced)
Powered by **LangChain, OpenAI, and ChromaDB**

- Semantic search across student data (RAG)
- AI-based performance prediction
- Attendance risk detection
- Class and department-level insights
- Multi-modal document support (PDF, images, docs)

---

## 🛠️ Tech Stack

- **Backend:** Python, Flask  
- **Database:** SQLite  
- **AI/ML:** LangChain, OpenAI, ChromaDB  
- **Data Analysis:** pandas, numpy, scikit-learn  
- **Document Tools:** PyMuPDF, python-docx, reportlab
